# PowerBI Folder

This folder contains the **Power BI Desktop file** for the Superstore Sales Analysis project.

## File

- `Superstore_Sales.pbix` : Interactive Power BI dashboard that visualizes key insights from the Superstore dataset.

## Dashboard Overview

The dashboard includes:

1. **Executive Summary**: Main KPIs and yearly sales trends.
2. **Customer Analysis**: Top customers, sales by segment, and geographic distribution.
3. **Product Performance**: Top products by sales and profit, sales by shipping mode, and product category performance.

## Notes

- The dashboard uses the processed fact and dimension tables generated from `Coding_and_Data_Modeling/Superstore_Data_Analysis.ipynb`.
- Ensure the `.pbix` file is opened in **Power BI Desktop** (latest version recommended) to view and interact with the reports.
